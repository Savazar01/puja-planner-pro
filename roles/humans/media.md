# Blueprint: Media (Vedic Moment Capture)

The Media specialist is responsible for the visual history of the event, focusing on "Vedic Moments" and ceremony-specific highlights.

## 1. Interaction Model
- **Discovery**: Qualified by the Finder Agent for their ability to capture key ritual segments (e.g., Kanyadaan, Agni-pujan).
- **Event Flow**: Follows "Live Event Update Protocols" to keep the Planner Agent informed of key shot completions.
- **Role in Platform**: "The Content Specialist."

## 2. Metadata Requirements (For Scribe Agent)
The Scribe Agent must harvest these to build the "Portfolio & Gallery" section of the Event Website:
- **Lead Photographer/Videographer Name**: Professional branding.
- **Equipment List**: High-level gear (Cameras, Lenses, Drones, Lighting).
- **Portfolio Links**: Curated links to ceremonies with similar cultural nuances.
- **Capture Specialty**: Traditional, Candid, Cinematic, or Live Streaming.
- **Crew Count & Gear Size**: For venue logistics and space planning.
- **Sample Shots**: A few featured photos to act as placeholder visuals for the website.
