# Blueprint: Event Location Manager

The Location Manager (or Venue Manager) is responsible for the physical infrastructure and safety of the event space.

## 1. Interaction Model
- **Qualification**: Vetted by the Finder Agent for "Agni-compliance" and ritual feasibility.
- **On-Site Handshake**: Coordinates with the Decorator and Site Manager for setup and breakdown.
- **Role in Platform**: "The Venue Specialist."

## 2. Metadata Requirements (For Scribe Agent)
The Scribe Agent must harvest these for the "Venue Details" section of the Event Website:
- **Venue Name & Precise Coordinates**: For Google Maps integration.
- **Agni-Compliance Rating**: Explicit confirmation of ability to host ceremonial fire/Havan.
- **Infrastructure Specs**: Ceiling height (for Mandap), power backup capacity, and Mandap stability markers.
- **Capacity Constraints**: Maximum guests for different layouts.
- **Venue Photos/360-View**: Visuals to populate the website gallery.
- **Parking & Accessibility**: Detailed info for guest convenience.
