# Blueprint: Pandit (Universal Practitioner)

The Pandit is the spiritual conductor of the ritual. This blueprint covers Ritualists, Astrologers, Ayurvedic experts, and Muhurtam specialists.

## 1. Interaction Model
- **Discovery**: Sourced by the Finder Agent based on "Service Category" (Rituals, Astrology, Ayurveda, etc.).
- **Scaffolding Handshake**: Collaborates with the Supplies Agent to define required Samagri.
- **Role in Platform**: "The Professional/Vedic Specialist."

## 2. Metadata Requirements (For Scribe Agent)
The Scribe Agent must harvest these to build the "Specialist Profile" on the Event Website:
- **Lead Specialist Name & Title**: E.g., Acharya, Shastri, Pandit.
- **Service Categories**: Rituals, Astrology, Ayurveda, Muhurtam.
- **Experience Summary**: Brief professional bio highlighting expertise.
- **Language Proficiency**: Sanskrit, Hindi, Telugu, etc.
- **Required Ceremonial Space**: Size of Havan Kund, seating requirements.
- **Consultation Availability**: Remote vs. Physical for pre-event Astrology/Muhurtam sessions.
